# Ren engine demo
This is sandbox application for my Ren engine.

## Linux build dependencies
- glfw3
- freetype2
- git
- meson

## Runtime dependencies
- OpenGL drivers installed. (3.3 core and ES support)

## Build & Run
    git clone https://github.com/TheRetikGM/Ren-demo
    cd Ren-demo
	./configure
	./compile
	build/src/main
